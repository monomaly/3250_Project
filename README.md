Project Overview
This project explores the complex relationship between digital technology habits and psychological health indicators. By analyzing a dataset containing metrics on screen time, sleep quality, stress levels, and mood, the goal is to identify patterns and predictive factors that contribute to or diminish mental well-being in a tech-driven society.

The project features a data-driven approach involving data cleaning, exploratory data analysis (EDA), and the development of an interactive diagnostic dashboard in Power BI.

Core Research Questions
Sleep & Stress: Is there a relationship between sleep duration and daily stress levels, and does this dynamic change across genders?
Screen Time & Sleep: Does higher daily screen time usage correlate with lower sleep quality?
Mood & Sleep: How does the quality of sleep experience directly impact an individual's daily mood rating?

Dataset Information
Source: Tech Use & Stress Wellness dataset (Kaggle).

Key Features:
Usage Metrics: Daily screen time, phone usage, laptop usage, and social media hours.
Well-being Indicators: Sleep duration, sleep quality (1-5), stress level (1-10), mood rating, and mental health scores.
Demographics: Gender and age.

Key Visualizations
The project uses Power BI to deliver the following insights:

Sleep-Stress Analysis: A Line Graph with Small Multiples showing the downward trend of stress as sleep increases, segmented by gender.

Screen Time Impact: A Scatter Plot with trend lines demonstrating the negative correlation between digital consumption and sleep quality.

Mood Correlation: A Column Chart illustrating the "staircase" effect of improving mood ratings as sleep quality increases.

Tech Used
